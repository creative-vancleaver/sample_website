Important HTML Tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Tags

Tag Name Purpose
h1–h6 Heading Include a heading (levels 1–6)
p Paragraph Include a paragraph of text
table Table Include a table
tr Table Row Include a row of data
th Table Header Make a table header
td Table Data Include a table data cell
li List Item Create new item in a list
header Header Creates header
blockquote Blockquote Creates a block style quotation
div Division Creates an unsytled divsion(block) of elements

Inline Tags

Tag Name Purpose Example Result
em Emphasize Make emphasized(italic) text <em>em</em> Technical sophistication
strong Strong Make strong(bold) text <strong>at least a billion people</strong> At least a billtion people
code Code Make monospace text <code>markup language</code> Markup language
a Anchor Make a hyperlink <a href="https://learnenough.com/">Learn Enough</a> Learn Enough
img Image Add an image <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl>
br Break Signals for a line break <br>Placed at end of line to prompt a new line</br> Prompts new line
Places new line here
span Span Allows inline text styling <span>Enables inline styling of contents</span> Enables inlne styling of contents
ol Ordered List Create new ordered(numbered) list <ol>My Top 3 Bands</ol>
    My Top 3 Bands
ul Unordered List Create new unordered(bulleted) list <ul>Concerts I've Been To</ul>
    Concerts I've Been To

Document Tags

Tag Name Purpose
html HTML Defines the document type as HTML
head Head Place to define Metadata
body Body Creates main Body of webpage
title Title Designates tab title of webpage
meta Meta Defines how web browser should read certain elements
