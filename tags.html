Important HTML Tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Tag Name Purpose
h1–h6 Heading Include a heading (levels 1–6)
p Paragraph Include a paragraph of text
table Table Include a table
tr Table Row Include a row of data
th Table Header Make a table header
td Table Data Include a table data cell
header Header Creates header
div Division Creates a division
Tag Name Purpose Example Result
em Emphasize Make emphasized(italic) text <em>em</em> Technical sophistication
strong Strong Make strong(bold) text <strong>at least a billion people</strong> At least a billtion people
code Code Make monospace text <code>markup language</code> Markup language
a Anchor Make a hyperlink <a href="https://learnenough.com/">Learn Enough</a> Learn Enough

span Span ?? <span>Call me Ishmael.</span> Call me Ishmael.
